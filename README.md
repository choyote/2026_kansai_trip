## 上傳／更新圖片的方法

例如要更新馬喰一代的「王道炭火燒套餐」：

1. 將新圖片命名為 `M002.png`。
2. 上傳到 `images/menu/`。
3. 覆蓋原本的 `M002.png`。
4. 不需要修改 `index.html`。

新增圖片時請使用下一個未使用的編號，不要重複使用或重新排列舊編號。即使某張圖片日後停用，也保留原編號，避免既有 HTML 連結失效。

## 圖片放大功能

餐廳菜單與地圖縮圖可點擊放大：

- 預設使用「適合畫面」顯示。
- 可按「原始尺寸」查看細節並捲動。
- 點擊圖片可在兩種尺寸間切換。
- 按 `Esc`、右上角 `×`，或點擊黑色背景可關閉。

若放大後顯示「圖片載入失敗」，請檢查檔名大小寫與路徑是否和圖片對照表完全一致。

## GitHub Pages 快取

若 GitHub Pages 暫時顯示舊圖，可強制重新整理，或等待約一分鐘後再查看。HTML 圖片網址目前附有 `?v=1`；若覆蓋圖片後快取長時間未更新，可把版本數字改成 `?v=2`。
